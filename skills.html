What I Have Learned So Far:

On this page you can see what I have learned in and around the world of Javascript. It hasn't been long at all and I am compiling an impressive list of areas of knowledge surronding Javascript and am sure that there is much more to come. There will never be a point when I have learned all that there is to know about web developement and that is truly exciting!

As an unremitting scholar, I am perpetually in search of ways to acquire and implement novel technologies.

The coding components that I have studied thus far:

HTML
Hyper-text-markup-language;

better known as HTML, represents the bottom most structure of code, very similar to a skeleton. The first div that I made and saw in an actual browser was a monumental moment for me that solidified my appreciation of coding and the process behind what we see.

CSS
Cascading Style Sheets;

better known as CSS, is the astheticly pleasing components on a page, like skin, hair, fingernail, etc. When we started CSS is when I started to see the true power that web developement posseses. Just the tip of the iceberg!

Bootstrap
Bootstrap Library ...

is a compilation of CSS-like style sheets that are pre-made and ready for consumption. This was my favorite of the three coding elements listed here in card form as it expanded on the HTML and CSS both and showed me that there is still much more to learn.

Javascript
Javascript

Javascript provides the functionality for your website. This is where your button clicks and inputs (and much more).

Servers
Servers...

The servers that I have worked with are Express and Mongo. These servers containg the routes that the data that we collect on the front-end and passes it back to be worked on.

DataBases
Databases...

Databases hold the data (so that our data persists beyond the session itself) that we are working on/with to be used. Specifically I have dealt with MongoDB as well as MysqlDB.

Publish
Publishing the site...

To publish (make your site public) my sites I used either GITHUB pages, Heroku or Netlify. I looked at what the publisher offered that was different than the other publishers so that I could tailor the publisher to my needs. There are other options to explore out there for publishing your work.

Virtual Studio Code
VS Code

VS CODE is the environment in which I run my code and test it for functionality and bugs along with modularization and user experience (in conjucntion with the developer tools in the browser). This is where I spent the overwhelming majority of my time learning coding. This is where you are able to actual code and put what you have been studying to practice.

React
React...

React is a library (similar to a framework like Angular or VUE) created by Facebook. It is a declaritive and flexible and efficient javascript library for building UI's (user interfaces are the part of the front-end of web developement where the user interacts with the site). One huge perk of using React is that it updates live (like running on a live server) so you don't have to stop and start your server in developement.
